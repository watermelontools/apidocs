# Get conversation replies

## Route

```
api/slack/getConversationReplies
```

## Method

POST

## Params

```
{
    "user": "string",
    "channelId": "string",
    "ts": number
}
```

Where:

- user: email of the user to search (registry on app). Not necessarily Slack email.
- channelId: channel id
- ts: the timestamp that Slack sends, found as `ts`

## Response

### Errors

If nothing comes in the `user` param

```
{
    "error": "no user"
}
```

If nothing comes in the `channelId` param

```
{
    "error": "no channelId"
}
```

If nothing comes in the `ts` param

```
{
    "error": "no ts"
}
```
