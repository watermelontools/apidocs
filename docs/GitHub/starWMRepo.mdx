# Star Watermelon Repo

## Route

```
api/github/starWMRepo
```

## Method

POST

## Params

```
{
    "user": "string",
}
```

Where:

- _user_: email of the user to search (registry on app). Not necessarily GitHub email.

## Response

### Errors

If the user does not exist, there will be no access token

```
{
    "error": "no access_token"
}
```

If nothing comes in the `user` param

```
{
    "error": "no user"
}
```

### Success

```
{"ok":"ok"}
```
