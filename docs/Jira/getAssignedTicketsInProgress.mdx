# Get Assigned Tickets In Progress

## Route

```
api/jira/getAssignedTicketsInProgress
```

## Method

POST

## Params

```
{
    "user": "string",
}
```

Where:

- user: email of the user to search (registry on app). Not necessarily Jira email.

## Response

### Errors

If the user does not exist, there will be no access token

```
{
    "error": "no access_token"
}
```

If nothing comes in the `user` param

```
{
    "error": "no user"
}
```
